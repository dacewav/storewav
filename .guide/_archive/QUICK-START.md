# ⚡ QUICK-START.md — Inicio Rápido (1 minuto)

> Versión compacta del mega prompt para cuando necesitas arrancar YA.
> Pega esto → la IA sabe qué hacer → no pierdes tiempo.

---

## Prompt Corto (copia y pega)

```
DACEWAV.STORE — co-piloto de desarrollo.
Stack: SvelteKit + Firebase + Cloudflare.

PASO 1: Ejecuta esto:
  cd store && bash .guide/BOOTSTRAP.sh

PASO 2: Lee estos 3 archivos (en orden):
  1. PROJECT_STATE.md → dónde estamos
  2. LIVE-STATUS.md → qué necesita atención
  3. BLOCK-CONTEXT.md → contexto del bloque actual

PASO 3: Dime qué bloque y qué tarea. Espera mi confirmación.

REGLAS:
- 1 tarea → testear → commit → siguiente
- No tocar bloques futuros
- No inventar features
- Si algo falla → arreglar antes de avanzar
- Al cerrar → REVIEW-GUIDE.md (actualizar guía)
```

---

## Prompt Ultra-Corto (si ya sabes dónde estás)

```
DACEWAV. Seguimos en [BLOQUE X]. Continúa con [TAREA].
Bootstrap ya hecho. Estado leído. Confirmo.
```

---

## Para Tareas Puntuales (interrupciones)

```
DACEWAV. Pausa el bloque actual.
Tarea puntual: [lo que necesitas]
Después de terminar, vuelve al bloque [X] tarea [Y].
No pierdas el progreso.
```

---

## Para Reparar Errores

```
DACEWAV. Error: [descripción del error]
No avances al siguiente bloque. Solo corrige esto.
Verifica que funciona antes de guardar.
```

---

## Para Avance Rápido (varias tareas seguidas)

```
DACEWAV. Avance rápido en Bloque [X].
Todas las tareas pendientes del bloque, sin preguntar entre cada una.
Solo párate si: falla un test, necesitas decisión de diseño, o terminas el bloque.
```
