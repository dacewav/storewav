## Uso en el sistema

Los contratos se usan en el flujo de compra:
1. Usuario selecciona beat → elige licencia
2. Se genera el contrato con datos del comprador + beat + fecha
3. Se envía por email como PDF adjunto post-compra
4. Ambas partes reciben copia firmada

## Formatos disponibles

- **Markdown** (`.md`) — editable, para generar versiones dinámicas
- **PDF** (`.pdf`) — versión final para firmar y enviar
