```
.store/
├── .guide/              ← Mega guide del proyecto (NO tocar)
│   ├── PROMPTS-LISTOS.md   ← Copiar y pegar en cada chat de IA
│   ├── PROJECT_STATE.md    ← Memoria compartida entre sesiones
│   ├── BLOCK-CONTEXT.md    ← Contexto del bloque actual
```

## Empezar

```bash
# 1. Clonar
git clone https://github.com/dacewav/store.git
cd store

# 2. Ejecutar bootstrap (instala skills, muestra estado)
bash .guide/BOOTSTRAP.sh

# 3. Abrir chat con IA y pegar el prompt default:
#    Copiar de .guide/PROMPTS-LISTOS.md → prompt 🟢 DEFAULT
```
