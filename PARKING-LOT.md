### Cómo usar

**Cuando tengas una idea:**
```
IA, guarda esta idea para después: [idea]
```
→ La IA añade a "Ideas Pendientes" sin interrumpir el bloque actual.

**Cuando estemos en el bloque correcto:**
```
IA, revisa el parking lot — ¿hay ideas para este bloque?
```
→ La IA revisa y sugiere las relevantes.

**Cuando decidamos no hacer algo:**
```
IA, descarta esta idea: [idea] porque [razón]
```
→ Se mueve a "Ideas Descartadas" con la razón.
